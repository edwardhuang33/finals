3. Breaks

= change in the probability distribution of the data

 

Breaks can be abrupt or gradual, often arising from changes in monetary policy

e.g. changing inflation target, interest rate changes for stabilisation

 

Breaks destroy external validity, so cause estimators to be biased

 

We can use the Chow test for breaks, in association with QLR if the break date is unknown (see 'Statistical Tests')

Other remedies:

o    If break date is known, split data in two and work separately

o    If break is artifact of outliers, remove those observations from sample

o    If break is result of regime shift, model shift using dummy variables

 

Note in general it is difficult to identify breaks as it can be hard to distinguish between multiple breaks and stochastic trends

4.3 STRICT EXOGENEITY & GLS

If we have strict exogeneity, any future, current or past value of X has no impacting on residuals

 

If this is the case, we can use a more efficient method to estimate dynamic causal effects known as Generalized Least Squares

 

But strictly exogeneity is difficult to justify in practice in many cases

Exogeneity should be evaluated on a case by case basis

e.g. Australian exports could be justified as exogenous with respect to US demand, as its impact on US demand is very small; EU exports would have to be endogenous as EU exports are large enough to significantly affect US income via a decline in EU income

 

Generally, exogeneity is not plausible if there is simultaneous causality, and strict exogeneity is not plausible if there is feedback
